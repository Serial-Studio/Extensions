# Solarized Dark

The iconic [Solarized](https://ethanschoonover.com/solarized/) dark color scheme, carefully adapted for Serial Studio.

## Features

- **Precision contrast.** Solarized's mathematically-derived contrast ratios ensure comfortable extended viewing.
- **Complete coverage.** All UI elements, dashboard widgets, console, and code editor are fully themed.
- **Code editor support.** Syntax highlighting uses the canonical Solarized palette (green keywords, blue types, cyan strings).

## Color Palette

| Role | Color | Hex |
|------|-------|-----|
| Background | Dark base | `#002B36` |
| Panel | Darker blue | `#073642` |
| Text | Light gray | `#EEE8D5` |
| Accent | Cyan | `#2AA198` |
| Highlight | Blue | `#268BD2` |
| Error | Red | `#DC322F` |
| Console | Green | `#859900` |

## Screenshot

This theme works best for long monitoring sessions where eye comfort matters. The warm-cool balanced palette reduces fatigue compared to pure-dark themes.
