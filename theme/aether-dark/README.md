# Aether Dark

Deep space dark theme inspired by GitHub's color palette. Cool blue accents over an ink-black canvas with soft gray text for long coding sessions.

## Features

- **GitHub-inspired.** A familiar dark palette drawn from GitHub's design system.
- **Cool blue accents.** Vibrant `#1f6feb` highlights and `#79c0ff` accent tones.
- **Green console.** Bright `#7ee787` terminal text for high readability against the dark base.
- **Full coverage.** Dashboard widgets, code editor, console, and all UI panels are fully themed.

## Color Palette

| Role | Color | Hex |
|------|-------|-----|
| Background | Ink black | `#0d1117` |
| Panel | Dark charcoal | `#161b22` |
| Text | Soft gray | `#c9d1d9` |
| Accent | Ice blue | `#79c0ff` |
| Highlight | Bold blue | `#1f6feb` |
| Error | Coral red | `#f85149` |
| Console | Mint green | `#7ee787` |

## When to Use

Ideal for developers who spend their day on GitHub and want a consistent visual language across their tools. The deep blacks and muted grays keep the focus on data, not chrome.
